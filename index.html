Projects

Surfscrape

A python script that uses the beautifulsoup library to scrape data on surf conditions from Surfline.com. When conditions at my home break are favorable it sends text message updates to me and my friends that include the current conditions and a motivational message. I used the apscheduler library to make the script run once every 24 hours indefinitely.
